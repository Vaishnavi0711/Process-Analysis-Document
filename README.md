# AI Process Consultant Assessment

## Candidate Submission

This repository contains my submission for the AI Process Consultant Assessment.

The assessment required analyzing an existing consulting firm's client delivery process, documenting the current workflow, identifying operational challenges, and recommending process improvements, automation opportunities, and AI-enabled enhancements.

The objective of this submission is not only to document the current process but also to demonstrate analytical thinking, business process understanding, stakeholder analysis, and the ability to transform an unstructured discovery conversation into structured documentation.

# Assessment Approach

The discovery transcript intentionally contains incomplete information.

Rather than making unsupported assumptions, this submission:

- Documents confirmed information separately from assumptions.
- Identifies information gaps requiring additional stakeholder discovery.
- Highlights operational risks.
- Provides practical recommendations for improving consistency and scalability.
- Identifies automation and AI opportunities that align with the firm's existing technology stack.
